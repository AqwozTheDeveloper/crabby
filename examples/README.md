# Examples

This directory contains example projects demonstrating Crabby's features.

## Basic Example

```bash
cd examples/basic
crabby install
crabby run index.ts
```

## TypeScript Project

```bash
cd examples/typescript
crabby install
crabby run src/app.ts
```

## Express Server

```bash
cd examples/express-server
crabby install
crabby start
```

## Dev Dependencies

```bash
cd examples/dev-deps
crabby install
crabby test
```
